Perguntas Frequentes

Que tipos de documentos o TechDoc consegue digitalizar e ler?

O TechDoc pode processar documentos como PDF, DOCX, imagens (JPG, PNG) e até documentos escaneados.

O que fazer se o documento está em baixa qualidade?

Nosso sistema utiliza tecnologia OCR (Reconhecimento Óptico de Caracteres) avançada, que melhora a leitura de documentos com qualidade reduzida.

O TechDoc consegue identificar texto em imagens?

Sim, ele extrai textos de imagens utilizando OCR, mesmo em fotografias de documentos.

Há limite de páginas para os documentos enviados?

Não, você pode enviar documentos de qualquer tamanho, mas o tempo de processamento será proporcional à quantidade de páginas.

Preciso fazer ajustes no documento antes de enviá-lo?

Não é necessário, mas recomendamos que os documentos estejam claros e bem alinhados para otimizar o reconhecimento.

Quais informações posso extrair de um documento?

O TechDoc pode extrair dados personalizados, como CPF, número de contrato, nomes ou outras informações específicas definidas pelo usuário.

Como selecionar as informações que quero extrair?

Durante o processo, o sistema pergunta quais dados são relevantes e adapta a extração conforme sua necessidade.

E se o documento estiver em um formato diferente dos suportados?

Você pode convertê-lo para um formato compatível, como PDF ou DOCX, usando ferramentas online ou o suporte TechDoc pode ajudar.

É possível ajustar os critérios de extração manualmente?

Sim, você pode configurar os critérios para buscar informações específicas dentro dos documentos.

O TechDoc oferece visualização do documento com as informações destacadas?

Sim, ele exibe o documento processado com os dados extraídos destacados para fácil verificação.

Preciso de conhecimentos técnicos para usar o TechDoc?

Não, o sistema foi projetado para ser intuitivo e de fácil uso, sem necessidade de experiência técnica.

O TechDoc pode ser integrado a outros sistemas?

Sim, o TechDoc oferece integração via API para conectar-se a outras plataformas.

Posso usar o TechDoc no meu smartphone?

Sim, ele é compatível com navegadores móveis e oferece a mesma eficiência em smartphones.

E se o sistema não conseguir processar meu documento?

Um relatório com detalhes do erro será gerado, e nosso suporte estará disponível para ajudar.

O TechDoc mantém uma cópia do documento após a digitalização?

Não, para segurança e privacidade, os documentos são automaticamente excluídos após o processamento.
